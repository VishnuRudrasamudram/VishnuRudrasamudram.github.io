AUTO IRRIGATION UNIT USING EMBEDDED SYSTEMS

The project aimed at achieving very effective automation at low cost in the field of agriculture. Dummy roots (tentiometers) are placed in various points in the field, which sense and interact with microcontroller for processing and automating irrigation.

  • Guided by: Naga Srinivasa Rao
  • Area: Embedded Systems

ANDROID DEVICE CONTROLLED TWO WHEEL SELF BALANCING ROBOT

An android application is developed to balance the LEGO robot. The android device, which has IMU, is kept on the robot to sense tilts. The device processes the data from sensors and gives control commands to balance the robot.

  • Guided by: Prof. C S Kumar
  • Area: Robotics, Control Systems

INDOOR AERIAL IMAGING USING MAV

Programmed a micro aerial vehicle using Robot Operating System (ROS) to autonomously travel and capture images of a specific area and stitched all images to get high detail mosaic view of that area. Language Used: Python

FACE TRACKING USING QUAD-ROTOR DRONE


The aim of this project was to make the drone to track the human face using its camera. We programed it to change its yaw according to the tracked face. The drone turns to centre itself with the detected face. It is programmed to track the face of the person closest to it. The project was made on Robot Operating System (ROS) using the Autonomy package. It was implemented on Parrot AR Drone quadcopter at Robotics Lab of RGUKT, R.K. Valley, Andhra Pradesh.

  • Guided by: Varun Kumar
  • Area: Robotics, Computer Vision
